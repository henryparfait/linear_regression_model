# Diabetes Progression Prediction – Linear Regression Task

**Mission:** Applying AI Tools in Healthcare  
**Problem:** Predicting diabetes disease progression one year after baseline using 10 clinical features (age, sex, BMI, blood pressure, and 6 blood serum measurements) from 442 patients.  
**Dataset:** Scikit-learn Diabetes Dataset (Efron et al., 2004)  
**Best Model:** Random Forest Regressor (Test MSE: 2835.96, R²: 0.46)

**API Endpoint:** https://diabetes-prediction-api-p7cy.onrender.com  
**Swagger UI:** https://diabetes-prediction-api-p7cy.onrender.com/docs
**Video Demo:** https://youtu.be/74LK4TLficM

## How to Run the Mobile App
1. Navigate to `summative/FlutterApp/`
2. Run `flutter pub get`
3. Run `flutter run`
4. The app connects to the live API endpoint above
